# Recipe Manager Web Application

## Overview
Recipe Manager is a full-stack web application that allows users to manage their personal recipes and meal plans.  
Users can sign up, log in, create recipes, and organize them into weekly meal plans through an intuitive web interface.

This project was developed as a practice of modern web application development using a React frontend, a FastAPI backend, and a PostgreSQL database.

---

## Features
- User authentication (Sign up / Log in)
- Create, view, and manage recipes
- Assign recipes to a weekly meal plan
- Persistent data storage using PostgreSQL
- Responsive and modern UI built with React and Tailwind CSS

---

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript (ES Modules)

### Backend
- FastAPI (Python)
- SQLAlchemy
- Pydantic
- JWT-based authentication

### Database
- PostgreSQL
